Regex is all about pattern matching and provides some nifty tools to filter out only values that contain information we know how to deal with.
Sometimes we might know exactly how the word starts and ends and might not really care about the middle part.
^ indicates the start of a word, so if I use ^abc then only words starting with letters abc would be matched. For ending, special character $ is used. Should I use abc$ then the word would need to end with said letters to qualify as a match.
Dot is a special case in regex and needs to be escaped, so if you wish to use it in your expression use \. instead.
There are many ways to communicate that you don't really care what character is next. \w for example would match any letter or digit, but would not match a dot. \S would match letters, digits and othe characters. \w and \S match a single character, to match multiple, you can add + so they become \w+ and \S+. NB! Our tests tuned so that regexp will not stop on first match and will check every single character whether it's a digit or not (regexp G flag is used).
