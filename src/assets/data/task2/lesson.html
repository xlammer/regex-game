Quantifiers in regular expressions define amount of matches of previous rule. For example expression a{2} for input string aabba will result in aa. If given input string doesn't contain double a's in the beginning then there will be no result. This should be easy, right?

Let's look at following table:
# Quantifier What it matches
1 {2} Exactly 2 symbols (may be any number)
2 {2,7} Between 2 and 7
3 {2,} 2 or more
4 * 0 or more
