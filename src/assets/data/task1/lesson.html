Let's start from the simplest rule, probably :) Matching a digit.
There are many cases in daily routines that require us working with integers that are for some reason trapped within a sequence of characters. Regex will help you find those digits.
In order to match a single digit you have to specify special character \d or use [0-9] as regular expression pattern. It will match any number form your input string.
NB! Our tests tuned so that regexp will not stop on first match and will check every single character whether it's a digit or not (regexp G flag is used).
