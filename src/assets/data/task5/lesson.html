The last task is about putting it all together. You should remember quantifiers and that they help matching exact number of symbols and setting boundaries. Click on 2nd task to check those out.

The first new expression for this task is alphabetical range. You may use range already in the first task, remember [0-9]? That is range that matches any digit from 0 to 9, but letters can be used as well i.e [a-z] will match any small letter from english alphabet and [A-B] will match only A or B (upper case).

The second new expression is conditional special character in brackets. (white|black) will match either black or white.
